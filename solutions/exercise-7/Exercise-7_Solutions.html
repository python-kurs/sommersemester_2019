Steps

  1. Get necessary data and inspect data and data structure.
  2. Get information for area definition and create it.
  3. Load the data
    • Create MultiScene from all files
    • Resample MultiScene to area definition
    • Blend the MultiScene to a timeseries
{::options parse_block_html="true" /} {::options parse_block_html="true" /}

Create area definition

{::options parse_block_html="true" /} {:.input_area} ```python area_id = 'Dem. Rep. Kongo' description = 'Dem. Rep. Kongo in Lambert Azimuthal Equal Area projection' proj_id = 'Dem. Rep. Kongo' proj_dict = {'proj': 'laea', 'lat_0': -3, 'lon_0': 23} width = 500 height = 500 llx = -15E5 lly = -15E5 urx = 15E5 ury = 15E5 area_extent = (llx,lly,urx,ury) area_def = AreaDefinition(area_id, proj_id, description, proj_dict, width, height, area_extent) ``` {::options parse_block_html="true" /}

Glob the files from the directory

{::options parse_block_html="true" /} {:.input_area} ```python file_dir = Path("../../data/msg") file_dir = Path("/media/droplet_data/python_course_data/exercise_7_data/") files = file_dir.glob("*20180*1200*.nc") ``` {::options parse_block_html="true" /}

Create the MultiScene

{::options parse_block_html="true" /} {:.input_area} ```python scenes = [Scene(reader="seviri_l1b_nc", filenames=[f]) for f in files] mscn = MultiScene(scenes) ``` {:.input_area} ```python mscn = MultiScene.from_files(files, reader="seviri_l1b_nc", group_keys=["processing_time"]) ``` {::options parse_block_html="true" /}

Load, resample and blend

Load the needed bands. Here we only load the 10.8 and 3.9 micrometer bands since we are going to use the difference of these channels for our cloud mask. Since we have multiple time steps we want to create a timeseries of these bands so we use the timeseries function we imported from satpy.multiscene as the blend function (the default function is stack which would use the first Scene as a base and fill gaps with susequent Scenes).

{::options parse_block_html="true" /} {:.input_area} ```python # Load the 10.8 micrometer band of all scenes within the MultiScene mscn.load(["IR_108", "IR_039"]) # Resample the MultiScene to our Greece domain new_mscn = mscn.resample(area_def, cache_dir=".") # Blend the scenes to one single Scene with each dataset/channel extended by the time dimension bscn = new_mscn.blend(blend_function=timeseries) ``` {::options parse_block_html="true" /}

Calculate the band difference

{::options parse_block_html="true" /} {:.input_area} ```python bscn["diff_108"] = bscn["IR_108"] - bscn["IR_039"] ``` {::options parse_block_html="true" /} {::options parse_block_html="true" /} {::options parse_block_html="true" /}

When we inspected our data in the first step we saw that there were also files from other time slots than 12:00. Since we want to compute the frequencies only for 12:00 we sort out these slots.

{::options parse_block_html="true" /} {::options parse_block_html="true" /} {::options parse_block_html="true" /} {:.input_area} ```python bscn["diff_108"] = bscn["diff_108"].sel(time=bscn["diff_108"].time.dt.hour == 12) ``` {::options parse_block_html="true" /}

Calculate the cloud mask

Using the band difference dataset of our Scene we calculated above and applying the threshold value we create the cloud mask.

{::options parse_block_html="true" /} {:.input_area} ```python bscn["mask"] = bscn["diff_108"] < -20 ``` {::options parse_block_html="true" /}

Now we can use xarray groupby functionallity to compute the monthly frequencies.

Up to now all processing steps were carried out lazy, meaning no calculation actually took place only the instructions were “recorded”. This is conveniant because due to the huge number of files we can not load all Scenes at once since they probably would not fit into memory.
To start the calculation we use the compute() method.

{::options parse_block_html="true" /} {:.input_area} ```python res = bscn["mask"].groupby("time.month").mean("time").compute() ``` {::options parse_block_html="true" /}
